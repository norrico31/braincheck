## Getting Started

### Dependency Tools

* React 18

### NOTE
* Node 18 above. 

### Installing

* Forked the repository 
```
git clone git@github.com:norrico31/braincheck.git
```
* cd into the repo
* Install node_modules
```
npm install --force
```
### Executing program
* To run the program
```
npm run dev
```
* To run build
```
npm run build
```
