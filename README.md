## Getting Started

### Dependency Tools

* React 18.2.0
* React Router DOM 6.22.3
* Framer Motion 11.1.7
* Axios 1.6.8

### NOTE
* Use node version 18.0 above 

### Installing

* First cd into the repo
* Install node_modules
```
npm install
```

### Executing program

* To run the program
```
npm run dev
```
* To build and generate html file
```
npm run build
```
